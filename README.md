# 心移植 術後管理ツール

**⚠️ 部外秘 — 関係者限定**

大阪大学心臓外科 成人心移植プロトコール（2020年12月改訂版）に基づく、スマホ向け術後管理 Web アプリ。

**URL:** https://ktonai103.github.io/tehaishi-tx-monitor/

---

## 機能

| 入力 | 算出される情報 |
|---|---|
| 移植日（必須） | POD・経過週数・該当プロトコール区分 |
| CNI種別（Tac/CyA） | トラフ目標値（**新機器換算済**）・旧値併記 |
| 体重（任意） | MMF 推奨量（<50 kg / ≥50 kg 分岐） |
| Ccr（任意） | バリキサ・デノシン腎機能別投与量 |

その他の固定表示：プレドニン漸減・次回生検日程（日付範囲）・併用薬チェックリスト

### ツール群（アコーディオン）
- 旧⇄新 換算機（Tac / CyA / EVL）
- Tac ↔ CyA 変換計算機
- CMV 用量（Ccr 依存）
- ステロイド漸減ガイド
- EVL 導入手順
- 拒絶反応治療プロトコル（Grade 別）

### リファレンステーブル
全 17 区分を新値で一覧表示、現在区分ハイライト

---

## 注意事項

- 本ツールは参考情報の簡易表示のみを目的とする
- 最終的な処方判断は主治医が行うこと
- 実際の処方は患者個別の状態により調節する
- プロトコール原典（斎藤俊輔/塚本泰正/久保田香 2020年12月改訂）を常に参照すること

## 新旧機器換算式

Y（新）= a × X（旧）+ b

| 薬剤 | a | b |
|---|---|---|
| タクロリムス | 0.964 | −0.029 |
| シクロスポリンA | 1.124 | 2.23 |
| エベロリムス | 1.519 | 1.498 |
