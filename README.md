# 心移植 術後管理ツール

**⚠️ 部外秘 — 関係者限定**

大阪大学心臓外科 成人心移植プロトコール（2020年12月改訂版）に基づき、
移植日を入力すると現時点のPOD・免疫抑制薬目標値（新機器換算済）・
生検スケジュール・併用薬チェックを自動算出するスマホ向けWebアプリ。

## 使い方

1. `index.html` をブラウザで開く（またはGitHub Pages URLにアクセス）
2. 移植日・体重・CNI種別を入力
3. 現時点の推奨値が自動表示される

## 注意事項

- 本ツールは参考情報の簡易表示のみを目的とし、最終的な処方判断は主治医が行うこと
- 実際の処方は患者個別の状態により調節する
- プロトコールの原典（斎藤俊輔/塚本泰正/久保田香 2020年12月改訂）を常に参照すること

## 換算式（新機器対応）

旧検査機器の目標値を新機器値に換算: Y（新）= a × X（旧）+ b

| タクロリムス | 0.964 | −0.029 |
| シクロスポリンA | 1.124 | 2.23 |
| エベロリムス | 1.519 | 1.498 |
